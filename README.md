### 🛠️ Technologies Used

This portfolio is built with:

- React.js ⚛️
- Vite ⚡
- Tailwind CSS 🎨
- Modern JavaScript 📜

### 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/kunalraj-313/my-portfolio.git

# Navigate to the project directory
cd my-portfolio

# Install dependencies
npm install

# Start the development server
npm run dev
```
